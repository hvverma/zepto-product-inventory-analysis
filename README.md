 Zepto Product Inventory Analysis

Project Overview
This project analyzes Zepto’s product inventory dataset using PostgreSQL to uncover insights related to product availability, pricing, discounts, and category performance.

Tools Used
* PostgreSQL
* Excel
* Kaggle Dataset

Dataset Information
* 3,700+ products
* Multiple categories and brands
* Inventory and pricing information

Key Analysis
* Category-wise product distribution
* Out-of-stock analysis
* Discount percentage analysis
* Pricing trends
* Brand-wise insights
* Inventory optimization recommendations

SQL Concepts Used

* SELECT, WHERE, ORDER BY
* GROUP BY and HAVING
* CASE Statements
* CTEs
* Aggregate Functions

Business Insights
* Identified categories with the highest inventory.
* Found highly discounted products.
* Analyzed out-of-stock patterns.
* Evaluated pricing strategies across categories.
